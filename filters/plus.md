---
title: plus
description: Liquid filter that adds a number to another number.
---

Adds a number to another number.

<p class="code-label">입력</p>
```liquid
{% raw %}
{{ 4 | plus: 2 }}
{% endraw %}
```

<p class="code-label">결과</p>
```text
{{ 4 | plus: 2 }}
```

<p class="code-label">입력</p>
```liquid
{% raw %}
{{ 16 | plus: 4 }}
{% endraw %}
```

<p class="code-label">결과</p>
```text
{{ 16 | plus: 4 }}
```

<p class="code-label">입력</p>
```liquid
{% raw %}
{{ 183.357 | plus: 12 }}
{% endraw %}
```

<p class="code-label">결과</p>
```text
{{ 183.357 | plus: 12 }}
```
